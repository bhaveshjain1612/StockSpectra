## In-Depth Stock Analysis

The "In-Depth Stock Analysis" section offers detailed insights into individual stocks:

- **Enter Stock Ticker:** Input the stock ticker in the sidebar, press Enter, and get insights for the chosen company.
- **Share Insights via URL:** Share company insights by passing the stock ticker through the URL query parameters.

Tabs for Comprehensive Analysis:

1. **Firmo Tab:** Shows firmographic data, sector, industry, and holdings chart.
2. **Stock Tab:** Presents key performance indicators (KPIs) and interactive charts with customization options.
3. **Financial Tab:** Provides financial metrics, both desirable and non-desirable, with detailed accessible data.

## Instructions

The "Instructions" section highlights the Financial Dashboard app's key features as tabs:

1. **Key Features:** Enables technical trend identification, deep stock analysis, financial statement access, and short-term trading opportunities.
2. **Data Sources and Updates:** Data is sourced from NSE and BSE, updated daily at 7:30 PM IST.
3. **How to Use:** Learn app navigation and utilization.
4. **Glossary:** Includes formulae for frequently mentioned technical terms.
